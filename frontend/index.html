How It Works

1

Create a Session

The host logs in and creates a broadcast session. A unique QR code and event code are generated instantly.

2

Start Broadcasting

Audio is captured as a raw PCM stream (16kHz mono) and streamed continuously to the server via WebSocket.

3

AI Transcription

Deepgram Nova-3 transcribes speech in real-time with built-in voice activity detection — zero hallucination, under 300ms latency.

4

Smart Translation

GPT-4o-mini translates each finalized transcript to every requested language with full context awareness.

5

Voice Synthesis

Deepgram Aura converts translated text to studio-quality audio on the server — first byte in under 250ms.

6

Live Delivery

Participants receive both text and audio via WebSocket. They see and hear the translation the moment the speaker finishes a sentence.
